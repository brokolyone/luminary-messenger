## üíª For Developers (Run Locally)

If you want to explore the code or run this project on your machine:

### 1. Download the Source
*   **Option A (Git):**
    ```bash
    git clone https://github.com/your-username/luminary-messenger.git
    cd luminary-messenger
    ```
*   **Option B (ZIP):**
    [Download ZIP Archive](https://github.com/brokolyone/luminary-messenger/archive/refs/heads/main.zip), extract it, and open the folder in your terminal.

### 2. Install Dependencies
Ensure you have [Node.js](https://nodejs.org/) (v18+) installed.
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
Open the link shown in the terminal (usually `http://localhost:5173`).

### 4. Build for Production
```bash
npm run build
```
